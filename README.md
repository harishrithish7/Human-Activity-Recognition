# Human-Acitivity-Recognition

Introduction
------------

Requirements
------------
1. MATLAB R2013a  
2. WEKA 3.7

Usage 
-----

####Installation

1. Download zip 
2. Unzip 
3. Open recognizer.m in MATLAB 

####MATLAB command line
```matlab
recognizer <weka_path> <video_name>;
```
####Example
```matlab
recognizer 'C:\Program Files\Weka-3-7\weka.jar' video.avi;
```
